MAIN.CSS
dentro da main.css os arquivos de cada import tem um ordem de hierarquia, de baixo pra cima.
Por exemplo, se repetir uma classe com mesma nome em um arquivo que esteja embaixo, é essa que vai ser aplicada. Voltando pra este projeto, o arquivo responsive.css vai sobrescrever todos os arquivos.
